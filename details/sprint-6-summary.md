# Sprint 6 — Database Fundamentals & Log Analysis

🔗 [View the full project repository](https://github.com/jamileazeredo/database-fundamentals)

This project introduced database querying and Linux terminal skills: analyzing server
logs using grep and directory commands, and writing SQL queries with COUNT, DISTINCT,
GROUP BY, HAVING, CASE, and JOIN to answer business questions from a ride-sharing
dataset. Feedback on this project reinforced the importance of using DISTINCT when
counting unique items, rather than relying on COUNT(*) alone — a subtle but important
distinction in query accuracy.
